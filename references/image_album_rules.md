# Image Album Rules

- Every image must appear in the final document or image album chapter.
- Use source file order unless the user gives an ordering rule.
- Generate captions automatically from sequence number and filename.
- For 1-3 images, insert near relevant content when possible.
- For many images, create an independent album chapter.
- Preserve aspect ratio and keep the subject visible.
- For portrait images, use one image per page or a restrained two-image page when space allows.
- Do not use hash filenames as formal captions. Use captions such as `图 11-1 贝贝灵熊猫 AI 伴侣形象图（一)`.
- Put raw image filenames in an appendix source table or material inventory.
- For the first stable version, do not put raw image filenames, dimensions, paths, width, height, or orientation in the formal document at all. Such details belong only in the internal report if needed.
