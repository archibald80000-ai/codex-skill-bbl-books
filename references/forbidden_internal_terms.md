# Forbidden Internal Terms

Formal DOCX/PDF/HTML output must not contain:

- 测试目录
- 清洗报告
- 处理报告
- 素材清单
- 原始素材
- 版本差异
- 相似度
- PackageNotFoundError
- error
- ~$
- width=
- height=
- orientation=
- paragraphs=
- tables=
- embedded_images=
- AI废话
- 删除多少条
- 归并多少条
- skill
- 脚本
- manifest
- 本次处理目标
- 本手册以测试目录
- 请使用浏览器或 PDF 工具生成目录页

These may appear in `BBL-books_内部处理报告.md` only when useful.
