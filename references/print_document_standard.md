# Print Document Standard

- A4 portrait.
- Margins: top, bottom, left, and right 2 cm.
- Chinese font: 宋体.
- Western font: Arial.
- Body: 12 pt.
- Heading 1: small-two or size-three equivalent, bold.
- Heading 2: size-four equivalent, bold.
- Table text: at least 10.5 pt, preferably 11 pt.
- Line spacing: 1.25-1.35.
- Cover has no page number.
- Table of contents is a standalone page.
- Body starts on a new page.
- Header contains only the document title.
- Footer page number is visible and normal.
- Tables must not be so narrow that text becomes vertical.
- Images keep aspect ratio and are not cropped.
- Figure captions must be formal and must not contain source filenames.
- No logs, paths, errors, or internal process notes in formal output.
